# PSI - MyPhotosApp

Trabalho realizado em contexto académico para a UC Projeto de Sistemas de Informação.

### Run Backend:
```
$ cd backend
$ npm install
$ SET DEBUG=backend:* & npm run devstart
$ DEBUG=backend:* & npm run devstart (MAC/Linux)
```

### Run Frontend:
```
$ cd frontend
$ npm install
$ ng serve -o
```
